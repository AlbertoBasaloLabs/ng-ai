# Environment Setup

```bash
# install and select node version
fnm i v22.14.0
fnm use v22.14.0
node -v
# install angular
npm i @angular/cli@19.2.22 -g
ng v
# create new angular project with ssr, routing and css
ng new astro-bookings --directory=./ -g --ssr --routing --style=css --experimental-zoneless -d
````
